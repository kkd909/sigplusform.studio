the studio

Built for environments where getting it wrong is expensive.

sig+form is a product design studio specializing in high-stakes data environments - legal, healthcare, and enterprise SaaS. We design and engineer interfaces where complexity is the constraint, not the excuse.

The work spans zero-to-one products, design systems, AI decision-support interfaces, and production frontend. Every engagement starts with the same question: what does the person on the other side of this screen actually need to do?

Verticals Legal · Healthcare · Enterprise SaaS
Shipped ~20 platforms across legaltech, healthtech, edtech
Stack Figma · React · TypeScript · Supabase · Anthropic API
Based Seattle, WA - available remotely
Funded 2 seed-stage products designed to close
